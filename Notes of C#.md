### 1. C\# and .NET

What is .NET?

.NET is just a platform where C# programs can run on it. It is like a JDK. It is not a compiler of C#, but incorporate a compiler of C#. 

### 2. Compiler of C#

The command of the compiler of C# is `csc`. It is in [csc](C:\Windows\Microsoft.NET\Framework\v4.0.30319\css.exe. ) of a Windows OS.  Add it to the Path of your operating system. 

Run it and an executable will be generated. (in Windows).

```shell
csc hello.cs
# hello.exe is generated in a Windows OS. 
```
